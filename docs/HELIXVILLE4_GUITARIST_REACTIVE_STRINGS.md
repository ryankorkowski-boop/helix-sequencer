# Helixville4 Reactive Guitarist

## Goal

Upgrade the Helixville4 snowman guitarist from static pose animation into a rhythm-reactive electric guitar performer.

The guitarist should visually communicate:

- chord strums
- pick attack direction
- neck slides
- riff energy
- sustain glow
- rhythmic groove
- chord intensity

- HX_SNOWMAN_GUITARIST_STRING_LOW_E

## Reactive Animation Behaviors

### Chord Sweeps

Down-strums should create horizontal sweeps across all active strings.

### Individual String Illumination

Lead lines should illuminate individual strings independently.

### Pick Attack Flash

Strong attacks should flash the pick zone and bridge.

### Neck Slide Tracking

Left hand position should visibly move across fretboard regions.

### Sustain Glow

Held notes should softly shimmer along active strings.

### Groove Motion

The guitarist body should subtly rock with rhythm timing.

## Helix Mapping Concepts

- chord density -> string count illumination
- velocity -> brightness and sweep thickness
- sustain -> shimmer duration
- pitch -> fretboard location
- palm mute -> shortened pulses
- lead riff -> single-string chase patterns

## Visual Target

The guitarist should feel energetic and rhythm-driven.

The approved target direction is the sheet-based reactive guitarist concept animation with visible strum sweeps, neck movement, and reactive guitar-string energy.
