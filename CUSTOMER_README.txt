Dream Sequence Weaver - Customer Quick Start
============================================

What this app does
- Generates xLights .xsq sequences from your audio, template XSQ, and layout file.
- Auto-detects model classes and places model-appropriate effects.
- Produces a single final all-models-aware output sequence per version for cleaner folders.

What you need
- Windows 10 or newer.
- xLights installed (to open/edit the generated .xsq files).
- Your own show files:
  - template .xsq
  - xlights_rgbeffects.xml (or .xbkp)
  - audio file (.wav preferred, .mp3 supported)

Fast workflow
1. Open DreamSequenceWeaver.exe.
2. Select Template XSQ, Audio Input, and Layout.
3. Click "Best Seller Preset" for the best-performing v20+ mix.
4. Click "Generate Sequence Set".
5. Open generated files in xLights and fine-tune if needed.

Output location
- Default output folder:
  Documents\DreamSequenceWeaver\Outputs

Support notes
- Use only sequences and assets you own or have permission to analyze.
- Review generated sequences before live show use.
